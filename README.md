# RentEase

RentEase is a broker-free rental platform aimed at simplifying the process of finding and listing rental properties. It enables direct interaction between property owners and tenants, reducing dependency on intermediaries and making the process more transparent.

This repository contains the frontend implementation of the application.

---

Features
- Responsive and modern UI
- Login and signup interfaces
- Property browsing and listing pages
- Property detail view
- Owner and tenant dashboards
- Add property flow (UI)
- Inquiry/interaction interface (UI)
- Reusable and component-based design

---

Tech Stack
- React (Vite)
- TypeScript
- Tailwind CSS
- shadcn/ui

---

Project Status

This project is currently a frontend prototype.  
It uses mock data to demonstrate application flow and user experience.
Backend integration (APIs, authentication, database) is planned for future development.
